<objective>
Replace the loosely-typed 8-key dict returned by `BatteryModel.get_convergence_status()`
with a frozen `ConvergenceStatus` dataclass. This tightens the contract between the
model and its three production callers (discharge_handler, virtual_ups_exporter, and
the model's own tests) and eliminates `.get("key", default)` lookups that mask typos
and missing fields.

Purpose: Type safety + immutability for a status object that already had a fixed
shape — make the contract explicit and enforceable.

Output: One new dataclass, one updated method signature, three updated call sites,
and updated test mocks/assertions. Net behavior identical; only the carrier type changes.
</objective>

Current return contract from `BatteryModel.get_convergence_status() -> Dict[str, Any]`:
```python
{
    "sample_count": int,           # Number of capacity measurements
    "confidence_percent": float,    # 0.0–100.0
    "latest_ah": Optional[float],   # Most recent ah_estimate, None if no samples
    "rated_ah": float,              # RATED_CAPACITY_AH constant (7.2)
    "converged": bool,              # len(estimates) >= 3 AND cov < 0.10
    "capacity_ah_measured": Optional[float],  # Locked baseline; None until first convergence
    "cov": float,                   # Coefficient of variation (0.0 if no samples)
    "mean_ah": float,               # Mean of ah_estimates (0.0 if no samples)
}
```

Target contract:
```python
@dataclass(frozen=True)
class ConvergenceStatus:
    sample_count: int
    confidence_percent: float
    latest_ah: Optional[float]
    rated_ah: float
    converged: bool
    capacity_ah_measured: Optional[float]
    cov: float
    mean_ah: float
```
