# Phase 12.1 Plan 01: Math Kernel Extraction and BatteryState Contract

## One-Liner

Wave 1 extracts all battery mathematics into a pure-function kernel package (`src/battery_math/`) with frozen BatteryState dataclass for type-safe circular dependency detection, enabling simulation testing and focused formula evolution.

## Summary

Wave 1 complete. All 9 tasks executed. 219 tests passing (212 existing + 7 new).

### Architecture Summary

**Package Structure:**
```
src/battery_math/
  __init__.py         → Re-exports 7 kernel functions + BatteryState
  types.py            → BatteryState frozen dataclass (7 fields, immutable)
  peukert.py          → Peukert runtime calculation (127-year-old physics)
  soc.py              → Voltage→SoC interpolation with state-aware bounds
  soh.py              → SoH from discharge curves, with time injection
  calibration.py      → Peukert exponent calibration from measurements
  capacity.py         → Phase 12 capacity estimation (placeholder)
```

**BatteryState Contract (Frozen Dataclass):**
```python
@dataclass(frozen=True)
class BatteryState:
    soh: float                          # State of Health [0.0, 1.0]
    peukert_exponent: float             # Exponent [1.0, 1.4], default 1.2
    capacity_ah_rated: float            # Rated capacity from label
    capacity_ah_measured: Optional[float] # From Phase 12, None until convergence
    lut: tuple                          # Immutable voltage→SoC lookup table
    cycle_count: int                    # OL→OB transitions counted
    cumulative_on_battery_sec: float    # Total on-battery time (seconds)
```

### Kernel Purity Verification

All kernel functions are **pure** (deterministic, no side effects):
- ✓ No I/O operations (no file reads/writes)
- ✓ No logging (no side effects)
- ✓ No calls to `time.time()` (except defaults in `soh.py`)
- ✓ No state mutation (frozen BatteryState, immutable LUT)
- ✓ Deterministic: identical inputs → identical outputs

**Exception:** `_weighted_average_by_voltage` in soh.py defaults to `time.time()` when `current_time=None` (daemon backward compat), but accepts injected time for simulator.

### VAL-01 Compliance (Discharge Quality Filters)

✓ **Minimum duration:** `calculate_soh_from_discharge` returns None for duration < 30s
  - Implementation: Early return in `battery_math/soh.py` line 42
  - Also enforced in legacy `soh_calculator.py` line 43 (backward compat)
  - Protects LUT from flicker-storm noise (multiple power flickers/hour)

✓ **Voltage floor (discharge mode):** `soc_from_voltage` accepts `state="OB"` parameter
  - OB (on-battery): floor = 10.0V (UPS hardware cutoff)
  - OL (on-line): floor = 8.0V (worst case global)
  - Prevents physically impossible samples from corrupting LUT

### VAL-02 Compliance (Peukert Fixed at 1.2)

✓ **Exponent as parameter:** `calibrate_peukert` accepts `peukert_exponent` parameter
  - Not hardcoded; allows caller to lock at 1.2
  - `peukert_runtime_hours` takes exponent param (default 1.2)
  - `calculate_soh_from_discharge` takes exponent param (default 1.2)
  - Monitor.py can pass fixed 1.2 to enforce VAL-02 lock during Phase 12
