Add structured journald event logging to MonitorDaemon._handle_discharge_complete() method for capacity measurements and baseline events.

Event types to log (standardized naming):
1. capacity_measurement: When CapacityEstimator produces a new Ah estimate
2. confidence_update: When convergence_score changes (not implemented yet, defer to v2.1)
3. baseline_lock: When convergence detected (CoV < 10% AND count >= 3)
4. baseline_reset: When user confirms new battery via --new-battery flag (already exists from Phase 13)

Implementation specifics:

**Event 1: capacity_measurement (after capacity_estimate obtained)**
Location: In _handle_discharge_complete(), after self.capacity_estimator.estimate() returns value
Trigger: Every time a capacity measurement is recorded to model.json

Log call:
```python
if capacity_estimate is not None:
    ah_estimate, confidence, metadata = capacity_estimate
    sample_count = self.capacity_estimator.get_convergence_status()[2]

    # Compute CoV for human-readable message
    estimates = self.model.data.get('capacity_estimates', [])
    ah_values = [e['ah_estimate'] for e in estimates]
    if len(ah_values) >= 2:
        mean_ah = sum(ah_values) / len(ah_values)
        std_ah = (sum((x - mean_ah) ** 2 for x in ah_values) / len(ah_values)) ** 0.5
        cov = std_ah / mean_ah
    else:
        cov = 0.0

    confidence_pct = int(confidence * 100) if confidence else 0

    logger.info(
        f"capacity_measurement: {ah_estimate:.2f}Ah (±{std_ah:.2f}), CoV={cov:.3f} "
        f"({sample_count} samples, {confidence_pct}% confidence)",
        extra={
            'EVENT_TYPE': 'capacity_measurement',
            'CAPACITY_AH': f'{ah_estimate:.2f}',
            'CONFIDENCE_PERCENT': str(confidence_pct),
            'SAMPLE_COUNT': str(sample_count),
            'DELTA_SOC_PERCENT': f'{metadata["delta_soc_percent"]:.1f}',
            'DURATION_SEC': str(int(metadata['duration_sec'])),
            'LOAD_AVG_PERCENT': f'{metadata["load_avg_percent"]:.1f}',
        }
    )
```

Field naming convention: SCREAMING_SNAKE_CASE for all custom fields (EVENT_TYPE, CAPACITY_AH, CONFIDENCE_PERCENT, etc.)

**Event 2: baseline_lock (when convergence detected)**
Location: After convergence check in _handle_discharge_complete() or in CapacityEstimator.estimate() return processing
Trigger: When get_convergence_status()[0] becomes True (first time CoV < 0.10 with count >= 3)

Log call:
```python
converged, measured_ah, sample_count = self.capacity_estimator.get_convergence_status()
if converged and not self.capacity_locked_previously:  # Track first convergence event only
    logger.info(
        f"baseline_lock: capacity converged at {measured_ah:.2f}Ah after {sample_count} deep discharges",
        extra={
            'EVENT_TYPE': 'baseline_lock',
            'CAPACITY_AH': f'{measured_ah:.2f}',
            'SAMPLE_COUNT': str(sample_count),
            'TIMESTAMP': datetime.now(timezone.utc).isoformat(),
        }
    )
    self.capacity_locked_previously = True  # Prevent duplicate logging
```

Note: Add instance variable self.capacity_locked_previously = False in __init__() to track if baseline_lock has already been logged

def test_journald_event_filtering(tmp_path, monkeypatch):
    """Verify journald events can be queried by EVENT_TYPE."""
    # Setup: Create mock journal handler
    captured_events = []

    def mock_send(**kwargs):
        captured_events.append(kwargs)

    # Patch the journald send function

        daemon = MonitorDaemon(...)
