    1. `soc_from_voltage(voltage: float, lut: list[dict]) -> float`
       - Takes normalized voltage and LUT from model
       - Returns SoC as decimal (0.0 to 1.0)
       - Algorithm: Binary search + linear interpolation
       - Edge cases: voltage > max (clamp to 1.0), voltage < anchor (clamp to 0.0)
       - LUT entries format: `{"v": voltage, "soc": soc, "source": "standard"|"measured"|"anchor"}`

       - Converts SoC decimal to percentage (0-100)
       - Example: 0.75 → 75
       - Return type: int (no decimals)

    Implementation notes:
    - Use binary search to find LUT bracket (v1 ≤ v ≤ v2)
    - Linear interpolation: soc = soc1 + (v - v1) / (v2 - v1) * (soc2 - soc1)
    - Handle edge case: voltage exactly matches a LUT point → return that SoC directly
    - Add logging for clamping (help debugging boundary behavior)
