<objective>
Complete calibration mode implementation by integrating cliff region interpolation into discharge completion event handling. When battery discharge completes (OB→OL transition) in calibration mode, automatically fill gaps in the cliff region and replace standard curve entries with measured/interpolated data.

Purpose: After operator manually discharges battery in calibration mode, daemon automatically processes collected data, interpolates the steep cliff region (11.0V–10.5V), and persists the improved model.json without requiring manual intervention.

Output:
- monitor.py calls interpolate_cliff_region() on OB→OL transition in calibration mode
- Model.json LUT updated with interpolated cliff region entries
- Daemon logs completion message directing user to restart without --calibration-mode flag
- Discharge buffer cleared after processing to prevent data reuse
- Unit test coverage for integration and edge cases
</objective>

   def test_interpolate_cliff_region_with_realistic_data():
       """Test interpolation with measured points from 2026-03-12 blackout."""
       # Setup: 3 measured points, 10 total after interpolation (3 measured + 7 interpolated)
       lut = [
           {'v': 13.4, 'soc': 1.0, 'source': 'standard'},  # Non-cliff, preserved
           {'v': 11.0, 'soc': 0.50, 'source': 'measured'},  # Cliff boundary
           {'v': 10.6, 'soc': 0.10, 'source': 'measured'},  # Cliff middle
           {'v': 10.5, 'soc': 0.0, 'source': 'measured'},   # Cliff anchor
       ]
       result = interpolate_cliff_region(lut)

       # Verify non-cliff entries preserved
       assert any(e['v'] == 13.4 and e['source'] == 'standard' for e in result)

       # Verify measured points present
       assert any(e['v'] == 11.0 and e['source'] == 'measured' for e in result)
       assert any(e['v'] == 10.5 and e['source'] == 'measured' for e in result)

       interpolated = [e for e in result if e['source'] == 'interpolated']
       assert len(interpolated) >= 7  # ~7 interpolated between measured points

       # Verify sorted descending
       assert result == sorted(result, key=lambda x: x['v'], reverse=True)

   def test_interpolate_cliff_region_removes_standard_from_cliff():
